### Требования к форме:

- поле **Username** обязательно для заполнения;
- поле **Password** обязательно для заполнения;
- поля **Username** и **Password** принимают данные только из списка, который расположен на странице;
- сообщение об ошибке появляется, если форма заполнено некорректно и имеет следующее содержание:
-
    - **"Epic sadface: Username is required"**, если поле Username не заполнено;
-
    - **"Epic sadface: Password is required"**, если поле Password не заполнено;
-
    - **"Epic sadface: Username and password do not match any user in this service"**, если в **Username** и/или *
      *Password** введены невалидные данные;
- сообщение об ошибке можно закрыть на крестик.

### Чек-лист проверок:

- [x] заполнить форму валидными данными из списка
- [ ] оставить поля Username и Password пустыми
- [x] заполнить поле Username, поле Password оставить пустым
- [x] заполнить поле Password, поле Username оставить пустым
- [x] заполнить поля Username и Password невалидными данными

### Требования к сортировке:

- Name (A to Z): сортирует товары по названию от A до Z;
- Name (Z to A): сортирует товары по названию от Z до A;
- Price (low to high): сортирует товары по стоимости от меньшей к большей;
- Price (high to low): сортирует товары по стоимости от большей к меньшей.

### Требования к сниппету карточки товара:

- изображение товара рендарится в размер 159 × 239 px;
- изображение является кликабельным -> переводит на страницу товара;
- название товара: color: #18583a; font-family: DM Mono, sans-serif; font-size: 20px; font-weight: 500. При наведение
  color: #3ddc91;
- название является кликабельным -> переводит на страницу товара;
- описание товара должно занимать <= 297 символов с учётом пробелов;
- нажатие кнопки "Add to cart" добавляет товар в корзину;
- нажатие кнопки "Remove" удаляет товар из корзины;
- после нажатия кнопки "Add to cart" она заменяется на "Remove";
- после нажатия кнопки "Remove" она заменяется на "Add to cart".

### Требования к иконке корзины:
- если товар в корзину не добавлен - иконка корзины отображается без счётчика (красный круг с цифрой);
- при добавлении товара в корзину на иконке появляется счётчик с кол-вом товаров в корзине;
- при изменении кол-ва товаров в корзине значение счётчика на иконке меняется;
- если все товары из корзины убраны иконка корзины остаётся без счётчика.
